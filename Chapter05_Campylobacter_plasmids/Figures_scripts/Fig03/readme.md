## Description

Figure 3 - Dendrogram based on MashTree results comparing  the sequences of 139 circularised plasmids (CPS). The coloured ranges show the 15 clusters identified according to their dissimilarity (Mash distance ≥ 0.009 and ≤ 90% of genes shared). The labels show the species and ST of each isolate. The filled triangles in the first ring indicate the source of each isolate. The plasmid markers and CC  are represented by the inner and outer colour strips, respectively. The red stars indicate the representative isolates selected for nanopore sequencing. The plasmid lengths (bp) are depicted in the bar charts in the last ring. The dendrogram was visualised and annotated using iTOL.

## Step 02
### Extract contigs of interest

Now we can run the following script that will extract the contigs of interest and save them to a new individual fasta file and add the ID name of the isolate onto the file name:

```bash

#!/bin/bash -e
#SBATCH --account=massey03742
#SBATCH --job-name=seqtk_input # job name (shows up in the queue)
#SBATCH --output=slurmlog/%x.%j.out
#SBATCH --error=slurmlog/%x.%j.err
#SBATCH --mem=5G            # RAM
#SBATCH --cpus-per-task=2 # number of cpus/threads, stick to 2 for now
#SBATCH --time=05:00:00 # Walltime (HH:MM:SS)

module purge
module load seqtk/1.5-GCC-12.3.0

# extract_contigs.sh
# Extracts specific contigs from fasta files based on a CSV lookup table

# --- Edit these paths ---
CSV="/nesi/project/massey03742/sequences_SACNZ/allsequences/contigs_list.csv"
FASTA_DIR="/nesi/project/massey03742/sequences_SACNZ/allsequences"
OUT_DIR="/nesi/project/massey03742/sequences_SACNZ/allsequences/circularisable_plasmids"
# -------------------------

mkdir -p "$OUT_DIR"

# Skip header line, read tab-separated columns
tail -n +2 "$CSV" | while IFS=$'\t' read -r ID CONTIG; do
    # Trim any trailing whitespace/carriage returns
    ID=$(echo "$ID" | tr -d '\r' | xargs)
    CONTIG=$(echo "$CONTIG" | tr -d '\r' | xargs)

    FASTA="${FASTA_DIR}/${ID}.fasta"

    if [[ ! -f "$FASTA" ]]; then
        echo "WARNING: $FASTA not found, skipping"
        continue
    fi

    OUTPUT="${OUT_DIR}/${ID}_${CONTIG}.fasta"

    # Use seqtk to extract the specific contig
    echo "$CONTIG" | seqtk subseq "$FASTA" - > "$OUTPUT"

    if [[ -s "$OUTPUT" ]]; then
        echo "Extracted: $ID -> $CONTIG -> $OUTPUT"
    else
        echo "WARNING: Contig '$CONTIG' not found in $FASTA"
        rm -f "$OUTPUT"
    fi
done

echo "Done!"

```
## Step 03
### Rename contigs headers

Now, let's just make sure to add the ID of the isolate into the contig's header inside the fasta file, to make it easier to identify each isolate in the mash analysis:

```bash
#!/bin/bash -e
#SBATCH --account=massey03742
#SBATCH --job-name=renaming_contigs_campy
#SBATCH --output=slurmlog/%x.%j.out
#SBATCH --error=slurmlog/%x.%j.err
#SBATCH --time=02:00:00
#SBATCH --cpus-per-task=2
#SBATCH --mem=5G

module purge
module load seqtk/1.4-GCC-11.3.0

# Directory containing the extracted contig FASTA files
input_dir="/nesi/project/massey03742/sequences_SACNZ/allsequences/circularisable_plasmids"

# Loop through all extracted contig FASTA files
for fasta in "$input_dir"/*.fasta; do
    # Extract the isolate name (everything before the first underscore "_")
    isolate_name=$(basename "$fasta" | cut -d'_' -f1)

    # Create a temporary file to store the modified FASTA
    temp_file="${fasta}.tmp"

    # Rename contigs by prefixing with the isolate name
    awk -v prefix="$isolate_name" '/^>/ {print ">" prefix "_" substr($0, 2); next} {print}' "$fasta" > "$temp_file"

    # Replace original file with the modified version
    mv "$temp_file" "$fasta"

    echo "Renamed headers in $fasta"
done

echo "All FASTA headers have been updated."

```
## Step 04
### Run Mashtree

Now with the contigs of interest saved in individual fasta files in a separate directory, we can run MashTree:

```bash

#!/bin/bash -e
#SBATCH --account massey03742
#SBATCH --job-name=mashdist_massey03742 # job name (shows up in the queue)
#SBATCH --output=slurmlog/%x.%j.out
#SBATCH --error=slurmlog/%x.%j.err
#SBATCH --time=24:00:00 # Walltime (HH:MM:SS)
#SBATCH --cpus-per-task=4 # number of cpus/threads, in our case 4
#SBATCH --mem=10GB            # Memory in MB

module purge
module load Mashtree/1.4.6-GCC-12.3.0

# Define the folder containing your plasmid FASTA files and the output file
input_folder="/nesi/project/massey03742/sequences_SACNZ/allsequences/circularisable_plasmids"
output_distances="/nesi/project/massey03742/sequences_SACNZ/allsequences/circularisable_plasmids/pcirc_campy_mashdistances.txt"
sketch_folder="/nesi/project/massey03742/sequences_SACNZ/allsequences/circularisable_plasmids"

#preparing the sketches to optimize run time
# Create a directory to store sketches if it doesn't exist
mkdir -p "$sketch_folder"

# Step 1: Create MASH sketches for each plasmid genome
echo "Creating sketches..."
mash sketch -o "$sketch_folder/plasmid_sketches" "$input_folder"/*.fasta

# Step 2: Calculate pairwise distances between all the plasmid genomes
echo "Calculating distances..."
mash dist "$sketch_folder"/plasmid_sketches.msh "$sketch_folder"/plasmid_sketches.msh > "$output_distances"

# Output completion message
echo "MASH distance calculation complete. Results saved in $output_distances"

# To plot the MashTree
mashtree --mindepth 0 --numcpus 12 *.fasta > mashtree_circ_contigs.dnd

```
## Step 05
### Plasmid markers classification

Using the plasmid typing scheme in ABRICATE, we are going to classify these contigs into plasmid types based on the combinations of plasmid markers they possess.

```bash
#!/bin/bash -e
#SBATCH --account=massey03742
#SBATCH --job-name=abricate_plasmids_massey03742 # job name (shows up in the queue)
#SBATCH --output=slurmlog/%x.%j.out
#SBATCH --error=slurmlog/%x.%j.err
#SBATCH --mem=5G            # RAM
#SBATCH --cpus-per-task=4 # number of cpus/threads, stick to 2 for now
#SBATCH --time=10:00:00 # Walltime (HH:MM:SS)


module purge

module load ABRicate/1.0.0-GCC-11.3.0-Perl-5.34.1

#LOOP

for fasta_file in /nesi/project/massey03742/sequences_SACNZ/allsequences/circularisable_plasmids/*.fasta; do
    base_name=$(basename "$fasta_file" .fasta)
    abricate --datadir /nesi/nobackup/massey03742/Linda_database/db/ --db CampyPlasmidtyping_abricate_20240710 "$fasta_file" > "${base_name}.tsv"
    echo "Finished processing $fasta_file"
done
```
