# Roary analysis

## Overview

This document shows the script used to run Roary. The assemblies generated with Nanopore long reads and Illumina short reads were annotated using Bakta.
The gff3 files only of plasmid contigs generated by Bakta were used as input to Roary.
Roary was used to compare the plasmid sequences gene presence/absence. The binary matrix generated by Roary is used to build a dendrogram that does not reflect any phylogenetic relationships. 
It is just a way to visualise the differences of all sequences using the gene presence/absence data.


### Roary script

```bash
#!/bin/bash -e
#SBATCH --job-name=roary_aline
#SBATCH --output=slurmlog/%x.%j_26sep25.out
#SBATCH --error=slurmlog/%x.%j_26sep25.err
#SBATCH --mem=15G
#SBATCH --cpus-per-task=4 
#SBATCH --account=massey03742
#SBATCH --time=24:00:00      # Walltime (HH:MM:SS)

cd /nesi/nobackup/massey03742/Campy_SACNZ/plasmid_annotations/gff_files

module purge
module load Roary/3.13.0-gimkl-2020a

#Run the script

roary -f /nesi/nobackup/massey03742/Campy_SACNZ/plasmid_annotations/gff_files/Roary_output -e --mafft -n -v -p 8 ./*.gff3

```
