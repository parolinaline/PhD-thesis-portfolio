# Scripts and other tutorials present in the Method section of my thesis

## Overview
| File/Folder | Description |
|-------------|-------------|
| [Assembly_method.md](./Assembly_method.md) | All the steps necessary to obtain the polished assemblies. It includes filtering, assembling and polishing |
| [Plasmid_typing_classification.md](./Plasmid_typing_classification.md) |Abricate script to obtain the plasmid typing scheme results |
| [RFPlasmid.md](./RFPlasmid.md) | RFPlasmid script used to obtain the prediction results for plasmid and chromosome contig classification |
| [MOB_suite_typer.md](./MOB_suite_typer.md) | MOB-suite typer script to predict mobility of plasmid sequences |
| [starAMR_script.md](./starAMR_script.md) | starAMR pipeline to detect point mutations associated with resistance using PointFinder database, detect AMR genes using ResFinder database, and confirm ST using MLST database |
| [cgMLST.md](./cgMLST.md) | Step-by-step tutorial on how I obtained the cgMLST of the whole-genome-sequences of *Campylobacter* plasmids |
| [quality_metrics.md](./quality_metrics.md) | All scripts used to assess the quality of Nanopore long reads and hybrid assemblies |
| [Roary.md](./Roary.md) | Roary script used to compare the gene presence/absence data of plasmid annotation files |
| [Plasmid_typing_CPS.md](./Plasmid_typing_CPS.md) | All steps involved in comparing plasmid sequences predicted as circularisable using Illumina long reads. This correlates to section 2.1.5 on my thesis. |
