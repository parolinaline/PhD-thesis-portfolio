# Welcome

This repository contains supplementary materials for my PhD thesis on bacterial genomics and antimicrobial resistance, with focus on *Campylobacter* and *Salmonella* surveillance in New Zealand.

## What you'll find here

Navigate by chapter to access all supporting materials for each section of my thesis:

- **R scripts** for phylogenetic analysis and visualization (ggtree, gggenomes)
- **Bash/Python workflows** for bioinformatics pipelines (chewBACCA, MLST analysis)
- **Step-by-step documentation** explaining methodology and intermediate steps
- **Raw data files** and processed results that support thesis findings
- **Supplementary tables and figures** too detailed for the main manuscript

## Quick navigation

- **[Chapters](chapters.html)** - Browse all thesis chapters and their materials
- **[About](about.html)** - Learn more about this project

## How to use this repository

Each chapter has its own dedicated section with:
- Overview of the research question and approach
- Links to relevant scripts and workflows
- Data files organized by analysis step
- Results, visualizations, and supplementary tables
- README files explaining each component

You can view code directly in the browser or download files for further analysis. All scripts are documented and ready to use.
